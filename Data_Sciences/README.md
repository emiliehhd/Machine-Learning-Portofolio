# Data Sciences 



## 📂 Contents

### 1. [Complete Data Science Project | FIFA22](https://github.com/emiliehhd/Project_DataScience)
**Goal:** A complete Data Science project applied to the FIFA 22 dataset (19 239 players), aimed at predicting the market value of players via a Gradient Boosting model and segmenting them into 5 separate profiles through KMeans clustering. The set is deployed as a REST API with FastAPI, containerized via Docker for simplified production.
* **Key Tasks:** Complete EDA, Supervised Models, Unsupervised Models.
* **Algorithms and Methods:** Regression, Clustering, KMeans, DBSCAN.
* **Tools:** `Pandas`, `Matplotlib`, `Seaborn`, `Scikit-learn`.
